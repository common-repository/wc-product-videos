== Description ==

This plugin allows you to replace your WooCommerce product images with a YouTube video.  The main product image, or any image within the product gallery can be replaced.

== Screenshots ==

1. Product page showing the main product image replaced with an embedded YouTube video
2. Product video admin entry on the edit product page allowing selection of video and position
